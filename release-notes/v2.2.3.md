# v2.2.3 Source Release

发布日期：2026-06-15

本次为 GitHub 源码更新：同步公开源码、文档和功能清单，不打包 Electron，不生成安装包 / blockmap / `latest.yml`，也不发布 GitHub Release 自动更新资产。

## 更新内容

- 动漫标签大师继续完善 Danbooru / Gelbooru 在线图库：在线源只作为 provider/source，分类保持画师 / 作品 IP / 角色 IP / General / Meta，不再把站点名称混入分类。
- 在线 tag 与作品分页保留懒加载模型，支持翻页、返回标签列表、搜索框回车触发、节点内与弹层内一致的搜索入口。
- 修复在线热门排序被前端通用搜索按名称重排的问题；Danbooru / Gelbooru 在线结果会保留后端按 `post_count` 返回的热门顺序。
- 收口 Gelbooru 无登录兜底：公开 autocomplete / HTML 兜底按 `post_count` 降序整理候选，封面失败不再用误导性占位图，预览请求会重试 3 次。
- 输出标签收敛为 `Tags:` 与 `Prompt:` 两行，避免输出素材节点出现冗余分类 / 属性文本。
- 同步修复紧凑 UI 中搜索按钮样式、按钮文字居中、长文本溢出、预览小眼睛与 100% 悬停预览。
- 公开版本同步到 `2.2.3`：`package.json`、`package-lock.json`、`vite.config.ts/js`、`electron/main.cjs`、`backend/src/config.js`、`features.json`、`README.md` 与本 release note 同步。

## 边界说明

- 本次只推公开源码，不运行 `npm run dist:release`、`npm run dist`、`npm run release:github` 或 `npm run release:verify`。
- 公开提交继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、Electron 缓存目录、维护者本地制作工具源码和任何私有商业能力 / 分组令牌实现。
- `comfyui-anima-t8/` 仅作为本机参考实现，不进入公开提交。
