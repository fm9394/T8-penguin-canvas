# v2.2.0 Source Release

发布日期：2026-06-14

本次为 GitHub 源码版更新，只同步公开源码、文档和功能清单；不打包 Electron，不发布 GitHub Release，不更新自动更新资产。

## 更新内容

- 新增「灵感之源」分类与「艺术风格大师」节点：迁移 qiaomu 艺术家风格库，支持英文名、中文名、流派和标签搜索，支持 lightbox、方向键切换、复制提示词、自定义分类/风格、导入导出，以及运行输出风格提示词或风格图片。
- 艺术风格大师第一版继续收口：节点缩略图改为内部滚轮浏览，长画家名和长分类不再撑出节点；输出按钮可直接生成对应 OutputNode；自定义风格支持本地上传图片并保存到用户风格库。
- 批量素材处理优先修复去除黑边能力：支持黑边、白边、透明边和自动检测，支持自动裁切、手动像素与 gap 类滑动调整，并在结果中显示裁掉的像素。
- Codex CLI Agent 简约模式与 Codex 生图工作台补齐上游图片预览、排序和删除，修复简约模式图像输出误落文本的问题，并保持创作台原有素材区逻辑不被破坏。
- 俄罗斯方块主题继续修复拖动节点顶部残影、大屏游戏模式布局、关卡速度与技能消耗曲线，并保留通关隐藏奖励与过场动画能力。
- 公开版本同步到 `2.2.0`：`package.json`、`package-lock.json`、`electron/main.cjs`、`backend/src/config.js`、`features.json`、`README.md` 与本 release note 同步。

## 边界说明

- 本次不运行 `npm run dist:release`、`npm run dist`、`npm run release:github` 或 `npm run release:verify`。
- 公开提交继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、Electron 缓存目录、维护者本地制作工具源码和任何私有商业能力 / 分组令牌实现。
- 真实 Grok OAuth 私有调用、New API 分组令牌等私有能力继续通过本机私有扩展 / 加密链路处理，公开仓库只保留安全外壳。
